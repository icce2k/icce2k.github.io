## Why UI Frameworks?
A User Interface (UI) Framework is the set of CSS classes and Javascript functions that help simplify the development of the web layout and provide an overall attractive look to a website. In addition, they do not require developers to have high technical and design skills since UI Frameworks provide some consistency across multiple browers and devices. However, it can be an even more daunting task to learn how to use one since its complexity can be compared to that of an entire new programming language. So, is it even worth the time and effort to basically learn a new language for web development when raw HTML and CSS do just fine?
While it can seem complicated at first, the time and effort you spend learning how to use a UI Framework definitely pays itself forward in the future. For one, they can help you gain momentum when you're just trying to get something on the page. This is because out of the box, they provide consistent and easy-to-use components that are great for developers who lack experience as well as thorough documentation that can come in handy when you're looking for a specific feature or forget classes. UI Frameworks are also open-sourced and are cross-compatible with different browsers which ensures that the code is reliable and tested. In general, UI Frameworks supply developers with a firm foundation for responsive design. Some popular UI Frameworks include React, jQuery, Twitter Bootstrap, and Semantic UI.
