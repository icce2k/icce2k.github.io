As our final project for ICS 111, my group and I created a game in Eclipse IDE where a player-controlled character must collect items to win and simultaneously avoid moving obstacles. Additionally, we programmed the obstacles to occasionally get the player's location and move toward it. After completing the project, we presented it to our class.

This was my first project and my first introduction into computer science. I learned the basics of programming and Java using the Eclipse IDE as well as what it is like to work on a program with others. I enjoyed this project and this class as I was able to express my creativity through programming and eventually spark my interest in computer science.

ICS 111 was my first introduction into computer science and it was what sealed my interest for this field. It was an introductory class that taught the fundamentals of computer science and programming such as problem solving, algorithm development, implementation, debugging, testing, and the basics of object-oriented programming. Despite having no experience in programming, I found this course to be fairly easy as I quickly caught on to Java and some of the aforementioned topics. However, a lot of the credit should also be given to the professor, Jason Leigh, and his well-designed course structure. At first, I naively thought that the rest of my academic career in computer science would be like ICS 111, but in the following semester, I shortly realized that computer science, like any other discipline, would take a lot more diligence and competitiveness to catapult yourself further into your career. The thought of this made me anxious for what lies ahead but it did not deter me and I am eager for the new challenges that await me.
